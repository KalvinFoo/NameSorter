<h1 align="center">Name Sorter</h1>
<div align="center">
Simple and powerful console application to sort a set of names by last name, then by any name the person may have.
</div>

### Usage :

Here is a basic example of using the name sorter.

1. Go to \bin\Debug\unsorted-names-list.txt and define a set of names. It can be as much as you want.
2. Save and close the file.
3. Run the application.
4. It should print the sorted list of names to your screen.
5. You can also find the sorted list in your local folder \bin\Debug\sorted-names-list.txt.
6. Done.
